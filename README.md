### 오늘 나는 무엇을 배웠는가?

삶을 살아가면서 계속 **오늘보다 나은 내일**을 만들 수 있다면  항상 발전하는 사람으로 있을 수 있겠죠.

하지만, 매일 이런 마음으로 삶에 임하는 것은 정말 쉽지 않은 일입니다.

그럴 때는 **이번 주보다 나은 다음 주**, **이번 달보다 나은 다음 달**을 목표로 해보는 것은 어떨까요?

훨씬 마음이 편해졌네요.

느려도 꾸준히 갑시다.
